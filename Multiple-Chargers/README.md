This Folder contains PYQT5 files for using 1, 2 or 3 Victron Solar Chargers in an ESS system.   
Required Equipment:   
Cerbo GX or RPI   
Multiplus II   
Smart Solar Charger (1, 2 or 3)    
SmartShunt
